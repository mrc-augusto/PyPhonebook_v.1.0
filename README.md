# PyPhonebook

A simple command-line phonebook application built with Python. Easily add, edit, delete, and manage your contacts, including marking favorites and viewing only favorite contacts.

## Features

- Add new contacts with name, phone, email, and favorite status
- View all contacts in a formatted list
- Edit existing contact information
- Delete contacts from the phonebook
- Mark or unmark contacts as favorites
- View only favorite contacts
- User-friendly menu and feedback messages
- Input validation and error handling

## How to Use
1. Clone the repository:
```bash
git clone https://github.com/mrc-augusto/PyPhonebook_v.1.0.git
```
2.Navigate to the project folder:
```bash
cd PyPhonebook_v.1.0
```

3.Run the application:
```bash
python phonebook.py
```

Menu Options

1. Add contact
2. View contacts list
3. Edit a contact
4. Mark/Unmark contact as favorite
5. View favorite contacts
6. Delete contact
7. Exit
